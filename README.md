# CS32-Final-Project
I hope to build a website that uses Python to monitor stocks, particularly those I already follow such as value stocks or high-quality businesses, while also having the ability to expand to others across the market. The system will track their recent price movements and flag stocks that experience significant fluctuations beyond a defined threshold. For each flagged stock, the program will gather relevant news  within the same time window and analyze repeated narratives and keywords across various sources using AI. The system will corroborate multiple pieces of information to generate a short and clear synopsis that explains the most likely reason for the stock’s movement. This should be a streamlined interface that investors can use to get be up to speed on market activity and quickly answers the key question of why the movement occurred.
