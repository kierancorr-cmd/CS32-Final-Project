# CS32-Final-Project
Through this project I hope to build a website that uses Python to scan and monitor stocks, particularly those I already follow such as value stocks or high-quality businesses, while also having the ability to expand to others across the market. The system will track recent price movements and flag stocks that experience significant fluctuations beyond a defined threshold. For each flagged stock, the program will gather relevant news articles within the same time window and analyze them using AI to identify repeated narratives and keywords across various sources. By corroborating multiple pieces of information, the tool will generate a short, clear synopsis explaining the most likely reason for the stock’s movement. The goal is to create a streamlined interface that investors can use to get be up to speed on market activity and quickly answers the key question of why the movement occurred.
